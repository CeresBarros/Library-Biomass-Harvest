Library-Biomass-Harvest is a support library for Extension Biomass Harvest.
Extension Biomass Harvest is found at:
(https://github.com/LANDIS-II-Foundation/Extension-Biomass-Harvest)


This repository contains the source code to build a Landis.Library.BiomassHarvest-v?.dll library.
The Landis.Library.BiomassHarvest-v?.dll is then placed in the Support-Library-DLLs repository.


The current version of Landis.Library.BiomassHarvest-v?.dll built by this repository is:
Landis.Library.BiomassHarvest-v2.dll.
